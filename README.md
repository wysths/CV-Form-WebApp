# CV-Form-WebApp
CV Form Web Application using Jakarta EE 10 with MVC Pattern developed by using NetBeans IDE and deployed on GlassFish server

---

## Features

- Generate a professional CV from a form
- Upload a profile photo
- Add education, work experience, skills, languages and reference
- View a hardcoded Example CV
- Print or Save as PDF

---

## Tech Stack

- Java 21
- Jakarta EE 10 (Servlets + JSP)
- Eclipse GlassFish 7
- Maven
- Tailwind CSS
- HTML + CSS

---

## MVC Structure

| Layer | File | Role |
|---|---|---|
| Model | CV.java | Stores all CV data |
| Controller | CVController.java | Handles form submission |
| Controller | ExampleCVController.java | Serves hardcoded example CV |
| View | myform.jsp | CV input form |
| View | processinput.jsp | CV output template |
| View | alert.jsp | Error page |

---

## How to Run

1. Clone or download this project
2. Open in NetBeans
3. Add GlassFish 7 under Tools → Servers
4. Right-click project → Clean and Build
5. Right-click project → Run
6. Open browser and go to:
